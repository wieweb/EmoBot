# Parts list

## Core Components
- ESP32-S3 DevKitC-1 (main controller / "Brain")
- ESP32-S3 Zero (head controller for display and servos)

## Actuators
- MG90S micro servo motor (metal gear, for future head rotation and tilt movement)

## Display
- 1.69" IPS LCD display (240x280 physical panel, ST7789V2 controller, SPI interface)

## Power Supply
- Miuzei USB-C power supply (5.1V / 5A, 27W, Raspberry Pi compatible)
- RUNCCI-YUN USB-C panel mount breakout (2-wire, 5V + GND input)
