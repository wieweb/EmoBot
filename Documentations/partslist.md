# Parts list

## Core Components
- ESP32-S3 DevKitC-1 (main controller / "Brain")
- ESP32-S3 Super Mini Development Board (head controller for LCD + servos)

## Actuators
- MG90S Micro Servo Motor (metal gear, for head rotation and tilt movement)

## Display
- 1.69" IPS LCD Display (240x280, ST7789 controller, SPI interface)

## Power Supply
- Miuzei USB-C Power Supply (5.1V / 5A, 27W, Raspberry Pi compatible)
- RUNCCI-YUN USB-C Panel Mount Breakout (2-wire, 5V + GND input)
